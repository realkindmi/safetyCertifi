Q.[서비스] 앱안심인증서비스란 무엇인가요?

인증받는 고객이 휴대폰을 직접 소지하고 있는지를 확인할 수 있는 휴대폰 소지 인증 서비스로서 고객의 인증 시도 시, 1차 고객의 인증문자 발송으로 진행하고 실패 시 2차 ARS를 통해 인증이 진행됩니다.

앱안신임증서비스가 보안에 더 강력한 이유는 기업에서 인증 코드 등을 발송하는 방식이 아닌 해당 고객의 휴대폰으로 보낸 암호화 메세지를 고객이 직접 문자 발송하는 방식으로 개발되어 스미싱걱정없이 더욱 안정적으로 인증이 가능합니다.
그래서 금융권 등 안전한 보안 관리가 필요한 서비스를 하는 기업들이 휴대폰 본인인증과 함께 휴대폰 소지인증(앱안심인증서비스)를 많이 이용하고 있습니다.

Q.[서비스] 앱안심인증서비스는 어떤 프로세스로 진행되는 건가요?

고객이 인증시도 → 휴대폰 암호화메세지 발송 → 해당 암호화메세지를 고객이 직접 발송하는 형태로 진행이 됩니다.
만일 문자를 통한 인증 실패 시, ARS로 2차 인증이 진행됩니다.

Q.[서비스] 본인명의의 휴대폰만 가능한가요?

휴대폰 소지 인증을 위한 서비스로 본인의 명의가 아닌 휴대폰이더라도 휴대폰을 소지하고 있다면 인증이 가능합니다.

Q.[서비스] 사용자가 직접 문자를 발송하는 이유는 무엇인가요?

기존 문자 수신 인증 방식은 기업이 사용자에게 인증 코드 등을 발송하는 방식이기에 스미싱 위험에 노출되어 있습니다.
앱안심인증서비스에서는 스미싱을 방지하기 위하여 사용자가 문자를 직접 발송하는 방식으로 채택하고 있습니다.

Q.[서비스] 서비스 신청은 어떻게 진행되나요?

앱안심인증서비스 홈페이지에서 사용하실 서비스를 구매하신 후 안내되는 계좌로 입금해주시면 됩니다.
입금확인이 완료되면 서비스에 적용하실 ClientID와 APIkey를 이메일로 발송해드리며, 관리자 사이트 내 개발가이드를 따라 개발적용 완료하시면 바로 이용 가능하십니다.

Q.[서비스] 트라이얼 버전은 무엇이며 어떻게 사용이 가능한가요?

홈페이지에서 트라이얼 신청을 클릭하여 가입하시면 트라이얼 버전 사용이 가능한 ClientID와 APIkey가 바로 발급됩니다. 트라이얼버전은 앱안심인증 문자 인증만 30일 동안 하루 10건씩 가능하며, ARS인증은 지원되지 않습니다.

Q.[서비스] 트라이얼 버전 사용 중 정식 서비스 변경은 어떻게 하나요?

앱안심인증서비스 홈페이지에서 원하시는 서비스를 선택 후 구매하시면 서비스변경이 됩니다.
트라이얼 버전에서 사용되던 인증키와 SDK파일은 트라이얼 사용 기간 이후로는 더 이상 사용할 수 없으니, 구매 후 새롭게 받은 인증키와 SDK파일로 변경해주시면 됩니다.

Q.[서비스] 월정액 서비스와 기간제 서비스는 무엇이 다른가요?

월정액 서비스는 매 월 서비스이용료를 지불하는 형태이며, 기간제 서비스는 6개월 혹은 1년 서비스 요금을 한꺼번에 결제하여 할인헤택을 더 받으실 수 있는 서비스입니다.

Q.[서비스] 서비스 시작은 언제인가요?

서비스 시작은 입금 확인 후 인증키 발송일입니다. 입금 확인 후 2일(영업일)기준으로 인증키가 발송됩니다.

Q.[운영] 서비스관리 및 서비스 이용현황, 통계 등 어디에서 확인이 가능한가요?

관리자 사이트에서 확인이 가능하며, 서비스 구매 시 입력하신 이메일아이디와 비밀번호로 로그인하시면 됩니다.

  • - 서비스관리 : 관리자사이트 홈 > 서비스관리
  • - 서비스 이용현황 : 관리자사이트 홈 > 이용현황
  • - 통계 : 관리자사이트 홈 > 통계

Q.[운영] 요금확인, 결제확인, 명세서, 증빙처리 여부 등은 어디에서 확인이 가능한가요?

관리자 사이트에서 확인이 가능하며, 서비스 구매 시 입력하신 이메일아이디와 비밀번호로 로그인하시면 됩니다.

  • - 요금확인 : 관리자사이트 홈 >실시간요금
  • - 결제확인, 명세서, 증비처리 여부 등 결제관련내역 확인 : 관리자사이트 홈 > 결제내역

Q.[운영] 보고서 생성이란 무엇인가요?

관리자 사이트 내에서 한달 사용 후 생성되는 보고서로 상세 사용 요금, 일별 문자인증 건수, ARS인증 건수에 대한 데이터를 PDF로 다운받을 수 있습니다.

Q.[운영] 청구서 알림 메일 변경이 가능한가요?

청구서, 혹은 사용량 알림 메일은 모두 이메일아이디의 메일주소로 전달 됩니다. 메일 주소는 관리자 사이트의 회원/결제정보 내에서 변경이 가능합니다.

Q.[운영] 서비스 변경은 어떻게 하는건가요?

이용하시던 서비스를 변경하고 싶은경우, 언제든지 변경하고 싶은 서비스를 구매 해 주시면 됩니다.
현재 이용중인 서비스의 이용기간까지는 해당 서비스로 계속 이용하시게되고, 새롭게 서비스가 시작되는 일자부터는 신규로 구매하신 서비스로 자동 변경되어 서비스를 이용하실 수 있습니다.

Q.[운영] 입금확인메일, 인증키발급 안내메일을 못 받으셨나요?

서비스 구매 시, 입력하신 메일로 안내 메일이 발송되니 입력하신 메일주소가 정확한지 확인해주세요. 메일주소가 정확하다면 스팸메일함으로 메일이 분류되었을 수 있습니다. 위 경우 모두 해당하지 않는다면 고객센터(02-6381-7509)로 연락주세요.

Q.[운영] 앱안심인증서비스는 몇 개의 앱에 적용이 가능한가요?

베이직을 제외한 서비스는 구매 후 서비스 적용은 무제한으로 가능합니다. 최초에는 1개만 사용하실 있도록 설정되어 있으므로 1개 이상 사용 하시려면, 관리자에서 앱서비스ID를 추가하여 사용해주세요.

Q.[운영] 서비스 해지는 어떻게 하나요?

관리자 사이트에서 해지가 가능합니다. 중도 해지는 불가능하며 남은 기간에 대한 환불은 이뤄지지 않습니다. 서비스가 해지되더라도 서비스 이용기간 마지막일 +7일까지는 관리자 사이트 접속이 가능합니다.

Q.[운영] 서비스 중지가 가능한가요?

구매하신 기간 내 중지는 불가합니다.

Q.[운영] 서비스 해지 후 동일 계정으로 다시 가입이 가능한가요?

서비스 해지 혹은 입금 미납으로 인한 중지 처리 시, 개인정보는 삭제되어 재가입하셔야 합니다.

Q.[결제] 서비스 결제 방식은 어떻게 되나요?

서비스 요금은 선불이며, 무통장 계좌입금 방식으로 예금주 ㈜시루정보 기업은행 002-420-5573으로 입금해주시면 됩니다.

Q.[결제] 청구서는 언제, 어디에서 확인이 가능한가요?

청구서는 사용 한달 후 다음날 청구되며, 이메일 아이디의 이메일 주소, 혹은 관리자 사이트의 결제내역 청구서에서 확인이 가능합니다.

Q.[결제] 결제일, 입금기한 언제인가요?

사용 한달 후 청구서가 발행되며, 청구서 발행 다음 날이 결제일이 됩니다. 결제일 7일 후 까지 입금이 되지 않으면 서비스가 정지됩니다.

Q.[결제] 증빙처리는 어떻게 요청 및 변경하나요?

개인인 경우 현금영수증 처리, 사업자인 경우에는 사업자번호 등록 시 세금계산서 발급이 가능합니다. 최초 구매 시 입력하거나, 이 후 에는 관리자 사이트에서 등록 및 변경이 가능합니다.

Q.[결제] 서비스는 과금방식은 어떻게 되나요?

기본 서비스 이용료는 선불 방식이며, 기본 서비스에 해당하는 문자인증 건수를 초과하는 경우, ARS인증에 대한 초과 요금은 후불로 청구됩니다.

Q.[결제] 서비스 이용료 연체 시, 이용이 불가한가요?

지정된 입금기한인 입금일 이후 7일이 지나며 서비스 이용이 불가합니다.

Q.[결제] 구매 후 환불이 가능한가요?

중도 해지가 불가능하여 환불은 불가능합니다. 구매하신 기간 마지막 날 까지 이용하셔야 합니다.

Q.[개발] 지원 대상은 어떻게 되나요?

단말 SDK iOS는 8.0 이상, 안드로이드는 4.0이상을 지원합니다. 서버 SDK는 JDK 1.6 이상을 지원합니다.

Q.[개발] 트라이얼 버전을 이용 후 정식 서비스 구매 시 다시 개발을 해야하나요?

트라이얼 이용 후 서비스를 구매 시, ClientID와 서버 SDK만 변경하여 사용이 가능합니다. 변경된 ClientID와 서버 SDK는 입금 확인 후 메일 혹은 관리자사이트에서 확인이 가능합니다.

Q.[개발] 서비스 개발적용은 어떻게 진행되나요?

서비스구매 또는 트라이얼 신청 후 이메일로 발송되는 인증키를 확인 해 주세요

개발 적용 시 문의사항은 관리자사이트 內 1:1 문의 또는 , 제품/기술 이메일 문의(appansim@seeroo.co.kr)를 통해 문의해주시면 됩니다.

관리자 사이트 內 개발가이드에 적용방법이 설명되어 있으니 참고하셔서 개발 적용하시면 됩니다.

Q.[운영] 서비스관리 및 서비스 이용현황, 통계 등 어디에서 확인이 가능한가요?

관리자 사이트에서 확인이 가능하며, 서비스 구매 시 입력하신 이메일아이디와 비밀번호로 로그인하시면 됩니다.

관리자 사이트에서 확인이 가능하며, 서비스 구매 시 입력하신 이메일아이디와 비밀번호로 로그인하시면 됩니다.

Q.[운영] 청구서 알림 메일 변경이 가능한가요?

청구서, 혹은 사용량 알림 메일은 모두 이메일아이디의 메일주소로 전달 됩니다. 메일 주소는 관리자 사이트의 회원/결제정보 내에서 변경이 가능합니다.

Q.[운영] 서비스 변경은 어떻게 하는건가요?

이용하시던 서비스를 변경하고 싶은경우, 언제든지 변경하고 싶은 서비스를 구매 해 주시면 됩니다.
현재 이용중인 서비스의 이용기간까지는 해당 서비스로 계속 이용하시게되고, 새롭게 서비스가 시작되는 일자부터는 신규로 구매하신 서비스로 자동 변경되어 서비스를 이용하실 수 있습니다.

Q.[운영] 앱안심인증서비스는 몇 개의 앱에 적용이 가능한가요?

베이직을 제외한 서비스는 구매 후 서비스 적용은 무제한으로 가능합니다. 최초에는 1개만 사용하실 있도록 설정되어 있으므로 1개 이상 사용 하시려면, 관리자에서 앱서비스ID를 추가하여 사용해주세요.
